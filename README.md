# ­ЪЇй№ИЈ Restaurant E-Commerce Platform (FullStack Project)

## ­ЪЊї рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ
рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ, рЃћрЃА рЃљрЃарЃўрЃА **рЃарЃћрЃАрЃбрЃЮрЃарЃюрЃўрЃА рЃћрЃџрЃћрЃЦрЃбрЃарЃЮрЃюрЃБрЃџрЃў рЃЎрЃЮрЃЏрЃћрЃарЃфрЃўрЃўрЃА рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА:
- рЃЊрЃљрЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃЊрЃюрЃћрЃю рЃЊрЃљ рЃњрЃљрЃўрЃљрЃарЃЮрЃю рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ;
- рЃЊрЃљрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃЮрЃю рЃарЃћрЃАрЃбрЃЮрЃарЃюрЃўрЃА рЃЏрЃћрЃюрЃўрЃБ;
- рЃЊрЃљрЃљрЃЏрЃљрЃбрЃЮрЃю рЃЎрЃћрЃарЃФрЃћрЃЉрЃў рЃЎрЃљрЃџрЃљрЃЌрЃљрЃерЃў;
- рЃњрЃљрЃљрЃцрЃЮрЃарЃЏрЃЮрЃю рЃерЃћрЃЎрЃЋрЃћрЃЌрЃљ рЃЮрЃюрЃџрЃљрЃўрЃю;


рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃўрЃљ **FullStack рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃќрЃћ** рЃЊрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃА Frontend-рЃАрЃљ рЃЊрЃљ Backend-рЃерЃў.

---

## ­ЪЏа рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

### ­Ъћ╣ Backend
- **JWT Authentication** Рєњ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ;
- **Swagger** Рєњ API Endpoint-рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ;

### ­Ъћ╣ Frontend
- **Angular (TypeScript, HTML, CSS)** Рєњ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃЊрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃў;
- **HttpClient** Рєњ API-рЃАрЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃў;
- **LocalStorage** Рєњ JWT рЃбрЃЮрЃЎрЃћрЃюрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃў;
- **Responsive рЃЊрЃўрЃќрЃљрЃўрЃюрЃў** Рєњ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃЊрЃљ рЃЊрЃћрЃАрЃЎрЃбрЃЮрЃърЃў рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА; 

---

## ­ЪћЌ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ
рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃўрЃДрЃЮрЃцрЃљ рЃЮрЃа рЃюрЃљрЃгрЃўрЃџрЃљрЃЊ:

- **Frontend (Angular)**  
  ­ЪЉЅ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў: рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ, рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ, рЃЏрЃћрЃюрЃўрЃБ, рЃЎрЃљрЃџрЃљрЃЌрЃљрЃерЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў.
- **Backend**  
  ­ЪЉЅ API рЃАрЃћрЃарЃЋрЃћрЃарЃў:  рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў, рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў, рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃў.  

рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ:  
```
Frontend (Angular)  <----->  Backend - API
```

## ­Ъџђ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### ­ЪЉц рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў
- рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ / рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- рЃАрЃџрЃљрЃўрЃЊрЃћрЃарЃў
- рЃЏрЃћрЃюрЃўрЃБрЃА рЃЊрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЉрЃљ
- рЃЎрЃћрЃарЃФрЃћрЃЉрЃўрЃА рЃЎрЃљрЃџрЃљрЃЌрЃљрЃерЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
- рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃњрЃљрЃцрЃЮрЃарЃЏрЃћрЃЉрЃљ

## РџЎ№ИЈ рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃў

### ­Ъћ╣ Backend
```bash
cd BackEnd
dotnet run
```

­ЪЉЅ `http://localhost:5294/swagger`

### ­Ъћ╣ Frontend
```bash
cd FrontEnd
npm install
ng serve 
```
рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃњрЃљрЃћрЃерЃЋрЃћрЃЉрЃљ:  
­ЪЉЅ `http://localhost:4200/`

## ­Ъј» рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ
рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА:
- рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Frontend рЃЊрЃљ Backend рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃљрЃЊ;
- рЃарЃЮрЃњрЃЮрЃа рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ JWT-рЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА;
- рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃарЃћрЃљрЃџрЃБрЃарЃў рЃарЃћрЃАрЃбрЃЮрЃарЃюрЃўрЃА рЃЮрЃюрЃџрЃљрЃўрЃю рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃўрЃА рЃљрЃгрЃДрЃЮрЃЉрЃљ;

рЃљрЃЏ рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљрЃЏ рЃЏрЃЮрЃЏрЃфрЃљ рЃфрЃЮрЃЊрЃюрЃљ рЃарЃЮрЃњрЃЮрЃарЃф **FullStack рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџ рЃюрЃљрЃгрЃўрЃџрЃерЃў**, рЃљрЃАрЃћрЃЋрЃћ рЃЌрЃћрЃЮрЃарЃўрЃБрЃџ рЃюрЃљрЃгрЃўрЃџрЃерЃў, рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃЋрЃћрЃЉ-рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃерЃў.
